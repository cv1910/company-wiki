# Hey Calendar Design-Erkenntnisse

## Kernprinzipien
1. **Wochenbasiert statt Monatsbasiert** - Menschen denken in Tagen und Wochen
2. **Timeline/Filmstrip-Ansicht** - Horizontale Zeitachse für den Tag
3. **Freetime hervorheben** - Ungebuchte Zeitblöcke sichtbar machen

## UI-Elemente
- **Day/Week Toggle** - Umschalter zwischen Tag und Woche oben links
- **Horizontale Tagesansicht** - Zeit läuft von links nach rechts
- **Farbcodierte Events** - Verschiedene Farben für verschiedene Kalender
- **"Sometime this week"** - Flexible Aufgaben ohne feste Zeit
- **Mehrtägige Events** - Balken über mehrere Tage

## Ansichten
1. **Tagesansicht** - Horizontale Timeline mit Stunden
2. **Wochenansicht** - 7 Tage nebeneinander, Events als Blöcke
3. **Jahresansicht** - Kompakte Übersicht aller Monate

## Design-Stil
- Helle, freundliche Farben (Lila, Blau, Grün, Orange)
- Abgerundete Ecken bei Events
- Klare Typografie
- Viel Weißraum
- Gradient-Hintergründe für Highlights

## Features für Company Wiki
- Urlaube als ganztägige Events anzeigen
- Eigene Termine erstellen
- Ganztägige/mehrtägige Termine
- Tag/Wochen/Monats/Jahresansicht
- Farbcodierung nach Typ (Urlaub, Meeting, etc.)
