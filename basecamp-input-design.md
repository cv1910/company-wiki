# Basecamp Chat-Eingabefeld Design

## Beobachtungen aus Screenshot

Das Basecamp Eingabefeld ist:
- **Einzeilig** - eine einfache horizontale Linie
- **Weißer Hintergrund** mit dünnem grauen Rahmen
- **Abgerundete Ecken** (border-radius)
- **Icons rechts**: Formatierung (A), Emoji, Anhang
- **Kein Placeholder-Text sichtbar** - sehr minimalistisch
- **Volle Breite** mit etwas Padding links/rechts
- **Fixiert am unteren Rand** des Chat-Bereichs

## CSS-Umsetzung
- Einfaches Input-Feld ohne Sprechblasen-Container
- Höhe: ca. 44-48px
- Border: 1px solid gray-300
- Border-radius: 8px oder rounded-lg
- Padding: 12px horizontal
- Icons als Buttons rechts im Input
