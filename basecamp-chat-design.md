# Basecamp Chat Design Analysis

## Key Design Elements from Screenshot:

### Message Bubbles:
1. **Other users' messages**: Beige/cream background (#F5F0E8 or similar warm beige), left-aligned with avatar on left
2. **Own messages ("Me")**: Light blue background (#D4E5F7 or similar), right-aligned with avatar on right
3. **Rounded corners**: Large border-radius (16-20px)
4. **No hard borders**: Soft, subtle appearance

### Message Layout:
- Avatar outside the bubble (not inside)
- Name + time in header: "Anna Hogrefe 10:06 •••"
- "Me" label for own messages instead of name
- Three dots menu (•••) for actions

### Reactions:
- Avatar + Emoji in pill-shaped container
- Multiple reactions shown horizontally
- Small circular avatar (24px) with emoji next to it
- Background color for reaction pill (subtle)
- "Add reaction" button with sparkle icon

### Date Separator:
- "TODAY" badge in center
- Amber/gold background (#F5A623 or similar)
- White text, uppercase
- Horizontal lines on both sides

### Colors to implement:
- Other message bg: #F5F0E8 (warm beige/cream)
- Own message bg: #D4E5F7 (light blue)
- Date badge bg: #F5A623 (amber/gold)
- Text: Dark gray/black
